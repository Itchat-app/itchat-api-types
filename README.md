# Zaun API Types

### Installation 
```s
$ npm i @zaunchat/types
```


### Example Usage
```ts
// for deno runtime 
// import type { APIMessage } from 'https://deno.land/x/zaunjs_types/mod.ts'
import type { APIMessage } from '@zaunchat/types'

const message: APIMessage = {
    id: 0,
    author_id: 0,
    channel_id: 0,
    content: 'Hi'
}
```
